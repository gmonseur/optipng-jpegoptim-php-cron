# optipng-jpegoptim-php-cron
  
Recursively optimize jpg and png images 


## Install image utils ubuntu
sudo apt install jpegoptim
sudo apt install optipng


## Usage
set work path in task_optipng_jpegoptim.php for example:

    $path = '/var/www';
